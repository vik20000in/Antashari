### Option 2: Use PowerShell to Download Sample Files

Run this command in PowerShell from the Antashari folder:

```powershell
.\download_sample_audio.ps1
```

This will download 21 sample royalty-free tracks automatically.

## That's it! 

Once the files are in the `audio/` folder, refresh your browser and the Tune Challenge will work!

## Notes

- Files must be MP3 format
- Keep filenames exactly as listed (lowercase with hyphens)
- Each file should be between 30 seconds to 3 minutes long
- Total size will be around 20-50 MB
